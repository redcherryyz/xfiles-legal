Privacy Policy

This Privacy Policy describes how the X Files Discord bot ("the Bot", "we", "our") collects, uses, and protects information when you use our service.

1. Information We Collect

Server Configuration Data: When a server administrator runs the /setup command, the Bot stores: the Discord server (guild) ID, the selected channel name font style, a mapping of channel names to channel IDs, a notification channel ID, and a detailed log channel ID. This data is necessary for the Bot to function.

Content Deduplication Hashes: The Bot stores cryptographic hashes (not the content itself) of files it has posted to prevent duplicate uploads. These hashes cannot be reversed to reconstruct the original content.

RedGifs Content IDs: The Bot stores RedGifs content identifiers it has already processed to avoid re-fetching the same content. These are public IDs from the RedGifs platform.

2. Information We Do NOT Collect

The Bot does not collect, store, or process: usernames or user IDs of server members (except the hardcoded bot owner ID for administrative commands), message content from users, direct messages, personal information such as email addresses or IP addresses, browsing habits or usage analytics of individual users, or voice channel data. The Bot does not read, log, or analyze any messages sent by users in Discord channels.

3. Data Storage

All data is stored locally on the Bot host machine in JSON files within the Bot's installation directory. No data is transmitted to external databases, cloud services, or third-party analytics platforms. Data is organized in per-server configuration files identified only by Discord guild ID.

4. Data Retention

Server configuration data is retained as long as the Bot remains in the server. When the Bot is removed from a server, the configuration data remains on disk but is no longer actively used. Content hashes are capped at 5,000 per channel per server and older entries are automatically pruned. RedGifs seen-content IDs are capped at 5,000 per content category.

5. Data Deletion

Server administrators may request deletion of their server's data by contacting the Bot owner. Removing the Bot from a server stops all data collection for that server immediately. The Bot owner can manually delete per-server configuration files upon request.

6. Third-Party Services

The Bot interacts with the following third-party services: Discord (via the Discord API, governed by Discord's own Privacy Policy) and RedGifs (for sourcing publicly available content, governed by RedGifs' Privacy Policy and Terms of Service). The Bot does not share any server or user data with these services beyond what is required for basic API functionality.

7. Welcome DM Feature

If the Bot owner enables the welcome DM feature, the Bot will send a pre-configured message to new members joining servers where the Bot is present. This message is identical for all users and does not involve collecting or storing any information about the recipient. Users who have DMs disabled will not receive this message.

8. Children's Privacy

The Bot is not intended for use by anyone under the age of 18. We do not knowingly collect data from minors. The Bot exclusively distributes adult content and should only be used in age-restricted Discord servers.
